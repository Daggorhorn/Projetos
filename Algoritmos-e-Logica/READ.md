
# ⚙️ Algoritmos e Lógica de Programação

## 📌 Sobre o repositório
Este repositório reúne exercícios e desafios resolvidos com foco no desenvolvimento da **lógica de programação** e construção de algoritmos eficientes.

Os problemas abordam desde conceitos básicos até intermediários, com foco em clareza, organização e boas práticas de código.

---

## 🚀 Conteúdos abordados

- Estruturas condicionais (if/else)
- Estruturas de repetição (for, while)
- Manipulação de variáveis
- Funções
- Vetores e matrizes
- Resolução de problemas computacionais

---

## 🧠 Objetivo
Desenvolver habilidades fundamentais para programação, como:

- Pensamento lógico
- Resolução de problemas
- Escrita de código limpo e organizado
- Interpretação de desafios

---

## 🛠️ Tecnologias
- Linguagem C

---

## 🎯 Diferencial
Cada solução foi desenvolvida com foco em:

- Clareza
- Eficiência
- Boas práticas
